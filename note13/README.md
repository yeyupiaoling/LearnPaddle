## data或code/data目录介绍
1. 该目录是存放上传文件的文件夹

## models目录介绍
1. `inference_topology.pkl`是预测的拓扑
2. `param.tar`是训练好的模型

## code代码介绍
1. `cnn.py`是卷积神经网络LeNet-5的定义
2. `index.html`是上传文件的网页
3. `infer.py`是测试预测的程序
4. `TestServer.py`是这个项目的主要程序，web的接口
5. `train.py`是训练模型的程序
